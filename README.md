## OPC-UA Client

An integrated OPC-UA data collection feature built directly into the database, eliminating the need for external data acquisition software.

## Key Features

- **Zero Dependency**  
  No additional runtime or external server required — fully built into the database  

- **Browse Support**  
  Select OPC-UA nodes for data collection with simple clicks — no coding needed  

- **Real-Time Data Processing**  
  Apply calculations and transformations during ingestion, enabling preprocessed data storage  

- **Decoupled Architecture**  
  Separate data collection and database servers to distribute system load and improve scalability
